  ## Getting Started with Shadcn

  This guide will walk you through setting up a new project with Shadcn, specifically using Next.js as the framework.
  
  Creating a new project
  Start by creating a new Next.js project. Use the create-next-app command as follows:

  ```bash
  npx create-next-app@latest my-app --typescript --tailwind --eslint
  ```

  Initializing Shadcn-UI
  Next, set up your project with Shadcn-UI. Run the following command:
  ```bash
  npx shadcn-ui@latest init
  ```

  Configuring components.json
  During the setup, you'll be prompted to answer several questions to configure components.json. This is crucial for project customization. ***Note: If you plan to integrate Storybook into your project, it's advisable not to use aliases for components and utils.*** 

  ```Would you like to use TypeScript (recommended)? no / yes
  Which style would you like to use? › Default
  Which color would you like to use as base color? › Slate
  Where is your global CSS file? › › app/globals.css
  Do you want to use CSS variables for colors? › no / yes
  Where is your tailwind.config.js located? › tailwind.config.js
  Configure the import alias for components: › @/components
  Configure the import alias for utils: › @/lib/utils
  Are you using React Server Components? › no / yes
  ```
